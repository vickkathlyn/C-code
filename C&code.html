programming
language

Uma linguagem de programação é um conjunto de regras e instruções que os programadores usam para criar programas de computador. É como uma linguagem que permite que os humanos se comuniquem com os computadores e especifiquem as ações que desejam que o computador execute. Os programadores escrevem códigos usando essa linguagem, que é então traduzida pelo computador para que ele possa entender e executar

Oq é linguagem C

“C” é uma linguagem de programação compilada com o propósito inicial de que fosse uma linguagem usada no desenvolvimento de uma nova versão do sistema operacional Unix, mas hoje é aplicada para criar softwares, criada em 1972, por Dennis Ritchie na empresa AT&T Bell Labs. A linguagem C existe desde antes da internet, é também muito usada em banco de dados para todos os tipos de sistemas: financeiro, governamental, mídia, entretenimento, telecomunicações, saúde, educação, varejo, redes sociais, etc. Grandes empresas como Apple, Microsoft, Oracle usam a linguagem C.

A linguagem C é conhecida por sua eficiência, flexibilidade e desempenho, tornando-a uma escolha popular para desenvolvedores de software e sistemas. Ela desempenhou um papel fundamental na revolução da computação e continua a ser relevante hoje.

Origem do nome

A linguagem foi chamada de C porque suas características foram obtidas a partir de uma linguagem anteriormente chamada de “B”, que de acordo com o Ken Thompson era a versão reduzida da linguagem de programação BCP

Quem foi o criador da linguagem C?

Dennis Ritchie foi um renomado cientista da computação e programador americano, amplamente reconhecido por seu papel fundamental no desenvolvimento da linguagem de programação C e do sistema operacional Unix. Ele nasceu em 9 de setembro de 1941, em Bronxville, Nova York, e faleceu em 12 de outubro de 2011.

Ritchie é mais conhecido por seu trabalho na Bell Labs (Laboratórios Bell), onde trabalhou ao longo de sua carreira. Ele foi um dos principais colaboradores na criação da linguagem de programação C na década de 1970. A linguagem C se tornou uma das linguagens de programação mais influentes e amplamente usadas na história da computação. Ela é conhecida por sua eficiência e portabilidade, sendo usada para desenvolver sistemas operacionais, aplicativos e software de baixo nível.

O trabalho de Dennis Ritchie na criação da linguagem C e no desenvolvimento do Unix desempenhou um papel crucial na expansão da computação e na evolução da indústria de tecnologia da informação. Sua influência perdura até os dias de hoje, e ele é amplamente considerado um dos pioneiros da computação moderna.

Aplicações na prática

Programa Olá Mundo;

Este programa é famoso pela simplicidade. Ela imprime a mensagem “Olá Mundo” no dispositivo (celular, notebook, etc.) Código usado:
/* Programa que exibe a frase "Oi mundo" na tela */
